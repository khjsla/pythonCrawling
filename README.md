# Python_Crawling
## Crawling Instagram Data with Python

<br>

### [HOW TO RUN]

```
python3 crawling.py
#OR
python crawling.py
```

<br>

### [HOW TO USE]

- YOUR_INSTA_ID
- YOUR_INSTA_PASSWORD
- YOUR_INSTA_CONTENT_NUMBER

위 세개의 변수를 통해 (조작을 통해)
<br>
원하는 계정의 <= ID + PASSWORD 
<br>
원하는 게시물의 데이터 크롤링 가능 <= CONTENT_NUMBER
